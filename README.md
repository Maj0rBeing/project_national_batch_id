# PROJECT_1-ATHLETES (ID Card Batch Generator)

## What you add
1) Put your template in the project root named: **id_template.png**
2) Put photos in: **photos/**
3) Edit the CSV: **id_data.csv**

## Install
Open PowerShell in this folder and run:
    python -m venv .venv
    .\.venv\Scripts\Activate.ps1
    pip install -r requirements.txt

## Run
    python main.py

Output files go to:
    output/
